# VS Code JSON interpreter for neovim

Work in Progress

## Installation

Python:
```bash
python3 -m pip install pynvim
```

packer:
```lua
 use({
    'k0nze/vscode_json.nvim',
     run=':UpdateRemotePlugins'
 })
```

## Configuration

lua:
```lua
require('k0nze/vscode_json.nvim').setup()
```
